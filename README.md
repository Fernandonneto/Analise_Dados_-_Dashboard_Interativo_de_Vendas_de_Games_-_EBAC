# 🧠 Análise de Dados: Dashboard Interativo de Vendas de Games

Este projeto faz parte do Módulo Looker Studio do curso de Cientista de Dados da EBAC e tem como objetivo a construção de um dashboard interativo no Looker Studio para análise de vendas no mercado de games.

A base de dados contém informações sobre vendas de jogos, incluindo categorias, gêneros, países e publishers, permitindo explorar padrões e identificar os principais fatores que impactam o desempenho comercial.

## 🧩 Etapas Desenvolvidas

**1. Importação da base de dados:** Carregamento e conexão da base no Looker Studio.

**2. Modelagem dos dados:** Organização das métricas e dimensões para análise.

**3. Construção do dashboard:** Desenvolvimento de visualizações interativas para facilitar a exploração dos dados.

**4. Criação de filtros dinâmicos:** Implementação de filtros por gênero, produto e categoria.

## 📊 Visualizações e Análises

- Vendas por país: \
Comparação do volume de vendas entre diferentes regiões.

- Vendas por categoria: \
Identificação das categorias com maior desempenho.

- Vendas por gênero: \
Análise dos gêneros mais populares no mercado.

- Vendas por publisher: \
Avaliação das empresas com maior participação nas vendas

## 📈 Resultados e Insights

- Identificação dos gêneros mais vendidos, com destaque para os que lideram em receita.

- Concentração das vendas em determinadas regiões, evidenciando mercados mais fortes.

- Presença dominante de alguns publishers, indicando concentração de mercado.

- Diferenças claras de desempenho entre categorias de jogos.

## ✅ Conclusão

O dashboard permite uma análise visual e interativa do mercado de games, facilitando a identificação de padrões e tendências. A utilização do Looker Studio demonstrou ser eficiente para transformar dados em insights estratégicos, apoiando a tomada de decisão baseada em dados.
